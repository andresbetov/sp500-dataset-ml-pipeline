Getting Started Guide - Guía de Inicio Rápido

Inicio Rapido (5 minutos)

1
Inicie la API FastAPI

Desde el terminal en la carpeta del proyecto:

cd packages/app && python -m uvicorn api:app --host 0.0.0.0 --port 8080 --reload
2
Sirva los archivos web

En otra terminal, desde la carpeta `web`:

cd packages/app/web && python -m http.server 8000

O con Node.js: npx http-server

3
Abra en el navegador

Visite: http://localhost:8000

✓ ¡Listo! Ahora puede explorar el dashboard, screener y análisis de tickers.

Paginas Disponibles

Dashboard

Resumen del mercado, estadísticas globales, top 10 tickers, performance del modelo

Screener

Tabla de 485 tickers filtrable, ordenable, paginable. Exportar a CSV.

Ticker Analysis

Predicción detallada, 29 indicadores técnicos, histórico 6m, exportación

Model Hub

Arquitectura del modelo, hiperparámetros, cross-validation, feature importance

Endpoints de la API

La aplicación web consume estos endpoints:

GET /api/health → Estado de la API GET /api/screener/summary → Resumen de mercado GET /api/screener?limit=50&offset=0 → Datos paginados GET /api/predict/AAPL?period=6mo → Predicción de ticker GET /api/model/info → Info del modelo

Configuracion

API URL: Configurables en `js/common.js`

CONFIG.API_BASE_URL = 'http://localhost:8080/api'

Preferencias del usuario: Guardadas en LocalStorage

settings.decimalPlaces = 4 // Precisión numérica settings.period = '6mo' // Período por defecto settings.theme = 'dark' // Tema (futuro)

🐛 Troubleshooting

⚠️ "Cannot fetch from http://localhost:8080"

Asegúrese que la API FastAPI está corriendo y CORS está habilitado en `api.py`

⚠️ "Network Error 404"

Verificar que el dataset.parquet existe en `data/processed/` y que los modelos están en `packages/ml/model/artifacts/`

⚠️ "Feature not found"

Ejecutar primero: `cd packages/ml/dataset && python main.py` para generar el dataset con todas las features
